# 其他备选路径详细分析

> 不是主推但值得了解的纯技术变现渠道

---

## 1. iOS App Store（纯 ASO 路线）—— 7/10

### 真实数据

- **Trading Tracker**（一个 solo 开发者）：$5K+（2022 4-12 月，纯 ASO 无营销）— [LinkedIn](https://www.linkedin.com/pulse/how-i-made-my-first-5k-indie-ios-developer-maros-petrus)
- **30-app 组合**：$22K/月（异常值）— [Indie Hackers](https://www.indiehackers.com/post/tech/from-failed-app-to-30-app-portfolio-making-22k-mo-in-less-than-a-year-myy3U7K9evxGOVOHti8s)
- **失败警示**：某 solo 8 个 app 只赚 $1,464 总收入

### 适合你的方向

- **日本本土工具**：家計簿、勉強時間管理、資格試験タイマー、漢字/英会話トレーナー
- **专业 niche tracker**：给小众职业群体（医生用药记录、设计师工时、厨师菜单）做精细日记 app
- **iOS 17+ Live Activity / Widget 类工具**

### 为什么不是主推

- 苹果 30% 抽成
- ASO 学习曲线陡（关键词研究 + 评分管理）
- 天花板比 Chrome 扩展低
- 但**你有 iOS 经验，作为副线值得做**

## 2. Mac App Store + Setapp 双发 —— 6.5/10

### Setapp 真实分成

- 70% 给应用开发者基于使用比例分配 + 20% 给推荐人，**最高 90%**
- 来源：[Setapp 文档](https://docs.setapp.com/docs/distributing-revenue) | [Setapp 开发者页](https://setapp.com/developers)

### 现实

- 总订阅用户基数十几万到数十万
- 独立 Mac 工具进 Setapp 通常每月几百到几千美元
- 难达到 $5K，**作为副收入很好**

### 适合的产品

- Mac 原生小工具：剪贴板增强、窗口管理、菜单栏小程序、PDF 处理
- 开发者向 Mac 工具：Git GUI 增强、log viewer、API 客户端

### 玩法

- App Store 走一次性买断（$15-$30）
- Setapp 走"被动副收入"（用户付订阅，使用比例分钱）

## 3. MQL5 Market（量化交易策略）—— 9/10（条件：有兴趣）

### 真实数据

- **Quantum Queen**：4.98/5、400+ 评论，最畅销 EA
- **Gold Reaper MT5**：~610 单 × ~$325K 累计
- **AI Gen XII EA**：~600 单 × ~$400K 累计
- 头部 EA 上线后稳定期月入 **$5K-$30K**
- 来源：[mql5.com market](https://www.mql5.com/en/market/mt5)

### 进入门槛

- **真实回测 + forward test 9-12 个月**才能登上榜
- 需要懂 MQL5 语言（C++ 风格）
- 策略真实有效才能持续卖

### AI 加速

- 策略想法用 Claude 做研究
- 写 MQL5 代码（Cursor 加速 5-10 倍）
- 跑回测 + 优化参数

### 风险

- 策略可能根本不有效 → 投入 1-2 月零产出
- 用户差评会淹没你的 EA

### 为什么不是主推

- **不是纯技术能解决的** —— 需要交易理解
- 时间窗口长（9-12 月才能上榜）
- 但**单条 ROI 最高**：一个有效 EA 月入 $10K+ 的概率不低于 Apify 头部

## 4. TradingView Paid Spaces —— 8/10

### 平台

- 平台费 15%（早鸟 0%）
- 需 Premium 账号
- 来源：[TradingView Creator Program](https://www.tradingview.com/blog/en/paid-indicators-strategies-on-tradingview-54934/)

### 数据

- 头部 Pine Script 作者订阅模式月入 $2K-$20K
- 2025 年新通道，窗口期较好

### 优势 vs MQL5

- Pine Script 比 MQL5 简单（Cursor 写起来更快）
- 用户基数更大（TradingView 有大量散户）

---

## 5. RapidAPI（卖 API）—— 7.5/10

### 平台

- 35,000+ APIs、4M 开发者
- 抽成 20-30%
- 来源：[Latka revenue](https://getlatka.com/companies/rapidapi)

### 赚钱模式

- 包装数据 API（金融、SMS、邮件验证等头部能上 $20K/月）
- AI API 中间层（GPT 包装、专门垂直 API）
- 爬虫即 API（与 Apify 重叠，可考虑两边都上）

### AI 加速

- 极强：Claude/Cursor 1-2 周可包装一个完整 API + 文档 + 测试

### 为什么不是主推

- 获客难（市场内 SEO + 关键词），所以"不需营销"打折
- 与 Apify 重叠度高，**优先 Apify**

## 6. Algora.io 编程悬赏 —— 7.5/10（验证仓位）

### 现状

- 累计支付 $65,785 给 188 名贡献者（48 国，2023 数据）
- 单条 bounty $50-$5000
- 头部贡献者一年 $20K-$50K
- 来源：[Algora docs](https://docs.algora.io/bounties/overview)

### 适合的角色

**不当主路径，当作"AI 加速验证"**：
- 1-2 周内即时现金流
- 验证"Claude Code + 你 20 年经验" vs 普通贡献者的速度差

### 玩法

1. 关联 GitHub
2. 盯 5-10 个活跃悬赏 repo（Tembo、Twenty、Million.js、Cal.com、Trigger.dev）
3. 用 Claude Code 一晚做 1-2 个中等 issue（$200-$500）
4. PR 描述用英文（AI 翻译够用）

### 局限

- 认领后 maintainer 接受率 30-50%
- 高竞争（很多人盯）
- 不可持续（不是被动收入）

---

## 7. Polar.sh 悬赏 —— 7/10

### 平台

- 平台费仅 5%（远低于 Algora）
- GitHub 官方支持
- 模式：issue funding + 订阅 + 数字商品
- 来源：[Polar.sh review](https://trybuildpilot.com/399-polar-sh-open-source-monetization-review-2026)

### 现状

- 还偏冷
- 单条 bounty $20-$500
- 需要自己有维护的开源项目才能拿到大单

### 与 Algora 区别

- Algora：贡献别人的 repo
- Polar.sh：你自己的 repo 收赞助/悬赏

## 8. Apify 之外的爬虫平台

### Datarade（B2B 数据中介）

- 准入难，多见结构化金融/地理数据公司
- 不适合独立开发者起步

### Kaggle Datasets

- 不直接付费
- 靠竞赛奖金（高门槛）

### 自建爬虫 SaaS

- 上 Gumroad / 自建站
- 流量获取难（绕回需要营销）
- **不如直接走 Apify**

## 9. JetBrains Marketplace —— 6.5/10

### 平台

- JetBrains 抽成最多 25%（**开发者拿 75%**）
- 用户基数小但付费意愿极强
- 来源：[JetBrains 文档](https://plugins.jetbrains.com/docs/marketplace/revenue-sharing-and-fees.html)

### 适合方向

- 后端/Java/Kotlin/Python 工具
- IDE 增强（代码生成、规范检查）

### 局限

- 用户基数比 Chrome Web Store 小 100 倍
- 没找到 solo 月入 $5K+ 的具体案例

## 已死/不推荐的路径

### ❌ Replit Bounties

2025/9/6 关停（[HN 讨论](https://news.ycombinator.com/item?id=44643875)）

### ❌ AI 内容站 + 广告变现

- 2024 年 3 月 Google HCU 后，独立站流量损失 30-90%
- 2025 年全球 publisher Google 流量下降 33%
- AI Overviews 出现的查询 zero-click 率 83%
- Mediavine/Raptive 主动拒绝 AI 内容站
- 来源：[Press Gazette](https://pressgazette.co.uk/media-audience-and-business-data/google-traffic-down-2025-trends-report-2026/)

**唯一变种可活**：日语技术 newsletter（beehiiv），但**不是 AI 内容站**

### ❌ GitHub Sponsors

- Caleb Porzio $112K/年靠 Laravel/Livewire 维护者身份 + Twitter
- 与"不出镜不品牌"画像冲突
- 来源：[Caleb Porzio Blog](https://calebporzio.com/i-just-hit-dollar-100000yr-on-github-sponsors-heres-how-i-did-it)

### ❌ VS Code Extension

- Microsoft 没内置付费机制
- 必须自建支付（Stripe/license server）
- 来源：[GitHub Issue 111800](https://github.com/microsoft/vscode/issues/111800)

### ❌ Raycast Store

截至 2025/2026，**无官方扩展付费分成**

### ❌ Shopify App Store

- 顶 25% 开发者 $14K/月（高天花板）
- 但**商家支持是日常英文沟通**
- 与画像不符
- 来源：[uptek 统计](https://uptek.com/shopify-statistics/app-store/)

### ❌ HackerOne 安全悬赏

- 需要专精 web 漏洞挖掘
- 与全栈/iOS 经验不直接对口
- AI 加速有限（Claude 对 0-day 挖掘帮助不大）

### ❌ WordPress Plugin

- 红海 + 老品牌垄断（EDD/WooCommerce/Gravity 10 年以上）
- 新人需要 $5K-$25K 初期投入做内容/SEO

### ❌ Hugging Face Spaces

无原生收入分成机制，主要是引流工具

## 路径组合策略总结

```
主仓位（70% 时间）：
└── Apify Store
    ├── 第 1 月：1 个 Actor
    ├── 第 6 月：MRR $1K-$3K
    └── 第 12 月：MRR $5K-$10K

副仓位（25% 时间）：
└── Chrome 扩展
    ├── 第 1 月：v1 上线
    ├── 第 6 月：MRR $500-$1K
    └── 第 12 月：MRR $3K-$5K（多扩展矩阵）

实验仓（5% 时间，前 4 周）：
└── Algora.io
    ├── 验证 AI 加速假设
    ├── 一次性 $500-$1500
    └── 验证完成后停止

可选探索（如有兴趣）：
├── iOS 日本本地 niche tracker（你有 iOS 经验）
├── MQL5 量化策略（高 ROI 但高门槛）
└── Mac App + Setapp 长期被动
```
