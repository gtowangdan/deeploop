# AI 应用方向

## 底层逻辑

用 AI 能力（调用API + 产品思维）解决具体问题，构建轻量级应用或工作流，卖给愿意付钱的用户或企业。

## 核心模型

1. **AI Wrapper 产品** —— 包装 Claude/GPT 能力，做垂直场景
2. **AI 自动化工作流** —— 替代重复人工，收订阅费
3. **AI 咨询 + 实施** —— 帮企业落地 AI，收项目费

## 当前机会窗口

- 大多数企业还不知道怎么用 AI
- 技术门槛在降低，产品思维是核心差异化
- 日本市场 AI 渗透率低，机会更大

## 研究待办

- [ ] 哪些垂直场景付费意愿最高
- [ ] 日本企业 AI 需求调研
- [ ] 最小可行产品的构建路径
