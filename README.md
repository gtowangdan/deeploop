# deeploop

> 自我认知 → 能力建设 → 搞钱方向 → 反馈回自我 —— 完整闭环

---

## 核心逻辑

```
认识自己
  ↓
盘清资产（技能/时间/人脉/资金）
  ↓
匹配搞钱路线
  ↓
执行 + 复盘
  ↓
重新认识自己（升级）
  ↑_______________↩
```

这个仓库是一个持续研究系统。每条路线都包含：底层逻辑、可行性分析、实施路线图、真实案例。

---

## 目录结构

```
deeploop/
├── self/               # 认识自己
│   ├── awareness/      # 自我认知框架
│   ├── skills/         # 技能盘点与评估
│   └── assets/         # 可用资产梳理
│
├── routes/             # 搞钱路线研究
│   ├── 01-coding-freelance/    # 技术接单
│   ├── 02-saas-product/        # SaaS 产品
│   ├── 03-content-creation/    # 内容创作
│   ├── 04-ai-applications/     # AI 应用
│   ├── 05-investing/           # 投资理财
│   └── 06-consulting/          # 咨询顾问
│
├── loop/               # 闭环机制设计
├── roadmap/            # 总路线图
└── README.md
```

---

## 使用方式

- 每个 `routes/` 子目录 = 一条完整的搞钱研究分支
- 每条路线包含 `README.md`（概述）、`logic.md`（底层逻辑）、`roadmap.md`（实施路径）
- `self/` 持续更新，是所有路线选择的基础
- `loop/` 记录复盘机制和迭代方法论

---

## 原则

1. **深度优于广度** —— 每条路线研究透，不追热点
2. **可执行优于理论** —— 有具体步骤，能落地
3. **闭环优于单点** —— 每个行动都有反馈回路
4. **认知是杠杆** —— 先想清楚，再下手
